# Co to za projekt?
W tym repozytorium został umieszczony projekt wykonany w ramach przedmiotu Bazy danych prowadzonego w Instytucie Matematyki (Uniwersytetu Wrocławskiego) w semestrze zimowym 2023/2024. 

# Co zostało wykorzystane w tym projekcie?
W tym projekcie wykorzystaliśmy poznaną wiedzę na temat PostgresSQL (w plikach został zawarty schemat zaprojektowanej bazy danych) i R (za pomocą pakietu shiny został stworzony interfejs graficzny) do stworzenia strony *Szkoły Delta*. Na stronie można zapoznać się z ofertą tej szkoły. Ponadto uczniowie i nauczyciele posiadają dodatkową funkcjonalność.
